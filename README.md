# ZenWares - Premium Fashion E-Commerce Website

ZenWares is a state-of-the-art, premium fashion e-commerce storefront designed for a modern shopping experience in Bangladesh. Built with clean semantic HTML, Bootstrap 5, custom Vanilla CSS, and modern interactive JavaScript.

---

## 🌟 Key Features

* **Responsive Design:** Optimized for all devices—from large desktops to small smartphones.
* **Premium UI/UX:** Curated HSL color palettes, elegant typography, smooth micro-animations, and clean layout patterns.
* **Compact Headers & Layouts:** Sleek `.shop-hero` page headers for a content-first browsing experience.
* **Interactive Quick View Modal:** View product details, select sizes/colors, adjust quantity, and add to cart without leaving the current page (optimized for mobile).
* **Fully Structured Directory:** Clean segregation of assets into dedicated folders.

## 📄 Pages Walkthrough

### 🏠 1. Home (`index.html`)
The main landing page of the website. It features hero banners, sliders, categories (Men, Women, Kids), popular collections, special deals, and testimonials.

### 🛍️ 2. Shop All (`shop.html`)
The product catalog page. Customers can view all available items and filter them by category, size, color, and price. It also includes sorting options and search capability.

### 🔍 3. Product Detail (`product-detail.html`)
A detailed description page for individual products. It features multiple image galleries, size/color selectors, quantity adjustment, detailed descriptions, and review tabs.

### 🛒 4. Shopping Cart (`cart.html`)
The shopping cart page where users can view their selected items, update quantities, remove items, see cart totals, and proceed to the checkout phase.

### 💳 5. Checkout (`checkout.html`)
The payment page where customers enter billing details, shipping details, and select their preferred payment method (Cash on Delivery or Online Payments).

### 🎉 6. Order Success (`order-success.html`)
The order confirmation page shown immediately after completing a checkout. It displays a summary of the order, transaction details, and a success message.

### 📦 7. Order Tracking (`track-order.html`)
An interactive order tracking page. Users can enter their Order ID to track the real-time shipment status of their purchases (e.g. Processing, Shipped, Delivered).

### ❤️ 8. Wishlist (`wishlist.html`)
A page listing all the products saved or liked by the user. Allows users to view their favorite items and directly add them to the cart.

### 👤 9. My Account (`my-account.html`)
The user's dashboard that displays account details, recent order history, order status overview, and navigation to edit profile settings.

### ✏️ 10. Edit Profile (`profile-edit.html`)
A user settings page where customers can update their profile information, name, email address, password, and shipping coordinates.

### 🔑 11. Auth Login (`login.html`)
A clean registration and sign-in page. Users can switch between the login form and registration form smoothly to access their accounts.

### 📝 12. Blog (`blog.html`)
A fashion blog and news page displaying articles, trends, guides, and stories to keep shoppers engaged.

### 🗺️ 13. Contact Us (`contact.html`)
The contact page containing physical store address, map details, contact info (phone/email), business hours, and a query submission form.

### ℹ️ 14. About Us (`about.html`)
The brand story page detailing ZenWares' vision, premium quality commitment, milestones, and introduction to the team.

### ❓ 15. FAQ (`faq.html`)
An interactive customer support page listing frequently asked questions about shipping, returns, payment, and security with a smooth accordion design.

### 🛡️ 16. Privacy Policy (`privacy-policy.html`)
A page detailing data collection, customer privacy protection measures, security protocols, cookies policy, and third-party disclosure terms.
